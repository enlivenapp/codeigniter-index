## History

<!-- History section to be expanded with community input -->

CodeIgniter has a long history as one of the most accessible PHP frameworks:

- **2006** — CodeIgniter v1 released by EllisLab, quickly gaining popularity for its simplicity and small footprint.
- **2011** — CodeIgniter v2 released, bringing improvements while maintaining backward compatibility.
- **2014** — CodeIgniter v3 released, continuing the lightweight tradition with incremental enhancements.
- **2015** — Ownership transferred from EllisLab to the British Columbia Institute of Technology (BCIT).
- **2020** — CodeIgniter v4 released as a complete rewrite, introducing namespaces, PSR compliance, a CLI tool, and modern PHP practices.

## Package Status Definitions

Each package in this catalog is assigned one of the following statuses:

| Status | Criteria |
|--------|----------|
| **Active** | Regular commits in the last 12 months, accepting issues/PRs |
| **Maintained** | Stable — receives security/compatibility fixes but no new features |
| **Archived** | Repo explicitly archived by owner or marked as end-of-life |
| **Unmaintained** | No meaningful activity for 2+ years, not explicitly archived |
